## Usage
### NPM
```sh
# npm
npm i web-cursor

# yarn
yarn add web-cursor

# pnpm
pnpm add web-cursor
```

```ts
import WebCursor from 'web-cursor'
```
### CDN
add script in html template
```html
<script src="https://unpkg.com/web-cursor@latest/dist/index.min.js"></script>
```
